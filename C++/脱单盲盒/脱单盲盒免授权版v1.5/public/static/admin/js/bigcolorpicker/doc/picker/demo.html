1、单个文本框注册取色板：

js代码：
	<input type="text" id="demo1Text"/>
	$(function(){
		$("#demo1Text").bigColorpicker();
	})
单击文本输入框弹出取色板，鼠标悬停或单击颜色小方块时将颜色HEX值设置在输入框的value上。
2、单个按钮注册取色板：

js代码：
	<input type="text" id="demo2Text"/><input id="bn" type="button" value="选色" />
	$(function(){
		$("#bn").bigColorpicker("#demo2Text");
	})
单击选色按钮弹出取色板，鼠标悬停在颜色小方块上时将颜色HEX值设置在输入框的value上。
$("#bn").bigColorpicker("#demo2Text") "demo2Text"为文本框id，也可以这么写：
$("#bn").bigColorpicker("demo2Text")
3、单个选择块注册取色板：

js代码：
	<div class="block-select"><a href="javascript:void(0)" id="bgImg" ></a></div>
	$(function(){
		$("#bgImg").bigColorpicker(function(el,color){
			$(el).css("backgroundColor",color);
		});	
	})
单击页面的选择块，鼠标悬停在颜色小方块上时将颜色设置为选择块的背景。
el:为<a href="javascript:void(0)" id="bgImg" ></a>
color:选择的颜色hex值。
$(el).css("backgroundColor",color)，通过回调函数设置a的背景颜色。
4、批量文本框注册取色板：

js代码：
	<input type="text" value="" name="textbatch"/>
	<input type="text" value="" name="textbatch"/>
	$(function(){
		$(":input[name='textbatch']").bigColorpicker(function(el,color){
			$(el).val(color);
		});	
	})
批量给页面上很多个文本框注册颜色取色板事件，将回调函数返回的颜色设置到相应的文本框中。
5、批量按钮注册取色板：



js代码：
	<input type="text" id="bn1"/> <input name="bnn" data-target="bn1" type="button" value="选色" />
	<input type="text" id="bn2"/> <input name="bnn" data-target="bn2" type="button" value="选色" />
	$(function(){
		$(":input[name='bnn']").bigColorpicker(function(el,color){
			$("#" + $(el).attr("data-target"))
			             .val(color);
		});
	})
批量给页面上很多个按钮都注册颜色取色板事件，通过注册单击事件元素的"data-target"自定义属性指向需要赋值的文本框。这里也是使用回调函数处理，比较灵活。
6、批量选择块注册取色板：

js代码：
	<div class="block-select"><a href="javascript:void(0)" name="bgImgBatch" ></a></div>
	<div class="block-select"><a href="javascript:void(0)" name="bgImgBatch" ></a></div>
	$(function(){
		$("a[name='bgImgBatch']").bigColorpicker(function(el,color){
			$(el).css("backgroundColor",color);
		});	
	})
批量给页面上很多个选择块都注册颜色取色板事件，将回调函数返回的颜色设置为a块元素的背景颜色。
7、自定义颜色块个数：

js代码：
	<input type="text" id="demo7Text"/>
	$(function(){
		$("#demo7Text").bigColorpicker("#demo7Text","L",3);	
	})
demo1Text：要设置颜色值的输入框id。
L：采用颜色区域展现使用li的实现方式（要可以改变格数大小，这个参数必须为L（l）。）
10:格数的基数，值在[2~10]之间。
