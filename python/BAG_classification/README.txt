1.存放路径不要有中文，良好的代码习惯从现在养成
2.本实验总共只有6个参数可以改，各位同学可以随意修改观察效果
3.本实验的数据路径可以根据实际文件存放位置进行修改（本实验原先用的训练和测试集均为证件照，原测试集存放路径为'./dataset/test/'，训练后准确率在85%以上。现在用的训练集为证件照，测试集为同学的照片）
4.本实验共分两个模型，rensnet.py为resnet18，resnet50.py为resnet50模型，前者层数较少训练较快，后者层数较多训练较慢
5.本实验每15轮会将learningrate缩小10倍，如假如第一轮为1，则第15轮为0.1，第30轮为0.01
6.若实验进行较慢，可以直接进行模型的加载然后对数据集进行测试
7.测试具体结果请看result文件夹

顺便本实验有个小彩蛋，有空的同学可以尝试找出来。(◍°∇°◍)
